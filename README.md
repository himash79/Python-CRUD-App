# Python-CRUD-App

## Requirements

01) Python 3.11.5
02) Pip 23.3.1
03) Postgresql 

## Project setup

01) Clone the project

		https://github.com/himash79/Python-crud-app.git
		
02) Open project using prefer IDE

03) Install dependencies

04) Configure the Database 

5) Execute the operations
